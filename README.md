# Algoritmos-SORT
Projeto desenvolvido no 4 semestre na disciplina de Projeto de Algoritmos que teve como objetivo a implementação e análise de algoritmos de ordenação com entradas crescentes, decrescentes e aleatórias. As instâncias variam de 10 a 1.000.000 de elementos. Os dados de entrada, saída e tempos de execução são armazenados em arquivos .txt, organizados automaticamente em pastas criadas pelo programa.
